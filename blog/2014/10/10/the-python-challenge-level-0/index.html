Level 0

Level 0的地址：http://www.pythonchallenge.com/pc/def/0.html

一张图片，上面写着2的38次方

Hint: try to change the URL address.

使用python求2的38次方的值：

import math
print(pow(2,38))

得到结果：274877906944

根据提示，修改url为：http://www.pythonchallenge.com/pc/def/274877906944.html。
